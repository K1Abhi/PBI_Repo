## Fielding Stats – Column Definitions (KPI)

| **Column_name**            | **Definition**                                                                                                                                                      |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Player**                 | Name of the cricketer.                                                                                                                                              |
| **Span**                   | The years between which the player appeared in matches (first year – last year).                                                                                    |
| **Mat**                    | Total number of matches played by the player.                                                                                                                       |
| **Inns**                   | Total number of innings in which the player fielded.                                                                                                                |
| **DisDescending**          | Total number of dismissals (fielding dismissals) credited to the player. May represent catches + stumpings.                                                         |
| **Ct**                     | Total number of catches taken by the player.                                                                                                                        |
| **St**                     | Total number of stumpings made by the player (usually applicable only if player kept wickets).                                                                      |
| **Ct Wk**                  | Catches taken **as a wicketkeeper**.                                                                                                                                |
| **Ct Fi**                  | Catches taken **as a fielder** (non-wicketkeeper).                                                                                                                  |
| **MD**                     | Match Dismissals – appears to be represented as a combined field format (example: "1 (1ct 0st)"). Shows total dismissals with breakdown into catches and stumpings. |
| **D/I**                    | Dismissals per Innings – calculated as total dismissals divided by number of innings fielded.                                                                       |
